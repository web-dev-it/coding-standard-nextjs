# Team Coding Standard

Це центральне місце для всіх правил написання коду в нашій команді.

## Як користуватися

- **Основний документ зі стандартом**: дивись розділ _Standard_ у меню нагорі.
- Використовуй цей сайт як **шпаргалку під час розробки**.
- Якщо бачиш, що якесь правило треба оновити — створи PR у цьому репозиторії.

## Структура

- **Standard** – повний опис стандарту коду
- (за бажанням) Можна додати окремі сторінки:
  - Code Review Guidelines
  - Git Workflow
  - Naming Conventions
